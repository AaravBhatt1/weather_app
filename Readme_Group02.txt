Interaction Design Group 2 Weather App

This repo hosts our mobile Flutter application, a weather app designed for gardeners in Cambridge. 

=== Prerequisites ===
1. To run it locally, you need the Flutter SDK, version 3.5.3 or newer.
2. You also need an Android or iOS simulator.
3. Finally, you need git (to clone this repository).

=== Setup and Installation ===
1. Install Flutter (if you haven't)
If you don't already have Flutter installed, download and follow the installation instructions from the Flutter website: https://docs.flutter.dev/get-started/install

2. Clone the Repository
git clone https://github.com/AaravBhatt1/weather_app

3. Navigate to the Project Directory
cd weather_app

4. Handle Dependencies
flutter pub get

5. Run the App
flutter run
